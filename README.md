# 챌린챌린

저탄소 생활을 챌린지 서비스를 제공하는 웹 애플리케이션.

챌린지는 저탄소 생활을 실천하고 이를 사진 또는 영상 인증을 통해 참여할 수 있다.

챌린지 참여 수, 좋아요 수에 따라 순위를 매겨, 랭킹을 보여준다.
